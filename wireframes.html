Design Note: Batch Upload Users select a content type, then upload multiple files at once. Each file gets a timestamp picker (since Google Drive strips EXIF). If multiple chat screenshots share the same timestamp, they are auto-grouped and stitching is attempted. Progress updates in real-time via WebSocket — showing upload %, then processing status (OCR, transcription, stitching).
Design Note: Smart Grouping When multiple chat screenshots are uploaded together, the system asks: "Stitch these into one conversation?" User can confirm, reorder, or exclude images before stitching.
Design Note: Animated Upload Zone When files are dragged over the upload zone, the dashed border animates with a rotating gradient effect (conic-gradient), providing clear visual feedback that the drop target is active.
